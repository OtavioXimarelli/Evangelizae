# Roadmap 9: Rebranding - Evangelizae

## Vision
Transform the platform from a personal prayer tracker ("Have You Prayed The Rosary Today") into a comprehensive missionary hub called **Evangelizae**. This shift focuses on digital witness, community missions, and apostolic sharing.

## Status: 🏗️ Implementation Phase
- **Core Identity:** Established in `src/config/brand.ts`.
- **Design System:** "Modern Cathedral" aesthetic defined with Glassmorphism and Gold Leaf accents.
- **Landing Page:** Narrative flow implemented with *Testimonium*, *Communio*, and *Missio* sections.

## 1. Brand Consolidation
- [ ] **Unified Typography:** Fully transition all components to *Cinzel* (Headers) and *Manrope* (Body).
- [ ] **Logo & Iconography:** Replace all "Rosário Vivo" mentions with the Evangelizae brand.
- [ ] **Multilingual Support:** Update `messages/` to reflect the new narrative in both PT and EN.

## 2. Interactive Experience (Narrative Flow)
- [ ] **GSAP Animations:** Implement ScrollTrigger for the landing page sections to create a high-end "Cathedral" feel.
- [ ] **Live Mission Stats:** Connect the *Communio* section to real-time data (Mock -> Real).
- [ ] **Apostolic Card Generator:** Build the functional image generator for the *Missio* section.

## 3. Community & Backend
- [ ] **Witness Submission:** Create the flow for users to submit their own *Testimonium*.
- [ ] **Circle Integration:** Link the "Power-up" mission blocks to actual user groups/circles.
- [ ] **Global Feed:** Implement the backend logic to aggregate public witnesses.

## 4. Visual Polish
- [ ] **Stained Glass Effects:** Refine glassmorphism utility classes for better accessibility and contrast.
- [ ] **Light Rays & Glow:** Enhance the `LightRays` component to provide a more sacred atmosphere.
- [ ] **Responsive Audit:** Ensure the narrative layout remains impactful on mobile devices.

## Timeline
- **Phase 1 (Visuals):** Completed (May 2026).
- **Phase 2 (Narrative):** Current Focus.
- **Phase 3 (Functionality):** Q3 2026.
